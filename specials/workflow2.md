# 公司项目时如何 push 代码？

同样也非常简单，按照以下步骤即可。

<br/>

### 步骤

-   把公司 repo 克隆到本地
    > 在本地打开 terminal，执行 `git clone`
-   开发完成后，在本地新建一个分支
    > `git checkout -b <branch_name>`
-   正常提交`pull request`
    > 不知道如何提交`pull request` ？ [点这里](./pr.md)
-   等待公司 mentor`review`

<br/>
