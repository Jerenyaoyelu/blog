# 概览

作为一种CSS扩展，不仅向后兼容CSS，它还使用现有的CSS语法新增了额外的特性。

# 变量

```less
@nice-blue: #5B83AD;
@light-blue: @nice-blue + #111;
#header {
  color: @light-blue;
}
```

输出：

```css
#header {
  color: #6c94be;
}
```

> 变量只能定义一次，实际上他们就是“常量”。

# Mixins（混合）



# 嵌套规则

- 它是一组css属性，允许将一个类的属性用于另一个类，并且包含类名作为其属性。
- 可以用类或者id选择器以与css样式同样的方式生命Mixin。
- 可以存储多个值，并且可以在必要时在代码中重复利用。
